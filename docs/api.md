# API Documentation

The API is self-documenting via **OpenAPI**, auto-generated by FastAPI.

- Interactive docs (dev only): http://localhost:8000/docs
- ReDoc (dev only): http://localhost:8000/redoc
- Raw schema: http://localhost:8000/openapi.json

Docs are disabled in production (`ENVIRONMENT=production`).

## Conventions (from Stage 1, Section 11)

- **Versioning:** URI-based, `/api/v1/...`.
- **Auth:** httpOnly session cookie (added in the Auth stage).
- **Error envelope:** every error returns
  `{"error": {"code", "message", "details?, "request_id"}}`.
- **Correlation:** every response carries an `X-Request-ID` header; the same id
  appears in logs and in the error body.
- **Status codes:** `400` validation, `401` unauthenticated, `403` not owner,
  `404` missing, `409` conflict, `422` semantic validation, `429` rate limited,
  `503` upstream unavailable.
